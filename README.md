# Proyecto-Integrador

Descripción: Repositorio del proyecto final integrador de la tecnicatura en Ciencia de Datos e Inteligencia Artificial, creado por la alumna Michell Andreina Zambrano Hereira.

Proyecto: Recomendador de Películas con Chatbot

Objetivo: Buscar ayudar a los usuarios a decidir qué ver, utilizando un chatbot que interpreta consultas en lenguaje natural.

Funcionalidades:

- Recomendación por género
- Recomendación por actor
- Combinación de filtros
- Consumo de API de TMDB

Tecnologías:

- Python
- Pandas
- Requests
- API TMDB

Cómo usar:

1. Clonar el repositorio
2. Instalar dependencias:
   pip install -r requirements.txt
3. Ejecutar:
   python src/main.py
